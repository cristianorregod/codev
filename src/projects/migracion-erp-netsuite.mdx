## Mi rol
Como desarrollador Fullstack, tuve un rol central en el diseño y ejecución de la estrategia de migración de datos desde un entorno Netsuite Legacy hacia Netsuite SuiteTax. 
Mi trabajo se enfocó en construir soluciones integradas, automatizadas y seguras para garantizar la transferencia exitosa de datos críticos.
Automatización de procesos: Desarrollé scripts en Python y flujos en Power Automate para automatizar la extracción, transformación y carga de información, reduciendo 
errores manuales y tiempos de ejecución.

- **Integraciones API REST y SOAP**: Implementé integraciones robustas con servicios de Netsuite y otros sistemas relacionados, tanto vía REST como SOAP, asegurando la 
correcta sincronización entre plataformas.
- **Consultas SQL optimizadas**: Elaboré consultas SQL complejas para extraer datos de distintas fuentes (bases de datos relacionales y archivos Excel), aplicando filtros, 
joins y transformaciones necesarias según los requerimientos funcionales.
- **Desarrollo backend**: Construí APIs con FastAPI que sirvieron como puente para que los bots de Power Automate interactuaran con el nuevo sistema de manera segura y eficiente.
- **Aseguramiento de la calidad de datos**: Implementé validaciones automáticas y controles de consistencia para garantizar la integridad referencial, la conciliación de 
saldos y la correcta estructura de los registros migrados.
- **Colaboración transversal**: Trabajé en conjunto con analistas funcionales, especialistas en Netsuite y el equipo de QA para asegurar que cada etapa de la migración 
cumpliera con los estándares de calidad y plazos definidos.

## Funcionalidades clave del proyecto:

- Extracción de datos desde múltiples orígenes (Excel, bases de datos relacionales).
- Normalización y transformación de estructuras según el esquema requerido por SuiteTax.
- Generación de APIs intermedias seguras y consumibles por bots de Power Automate.
- Validación de integridad referencial y conciliación de saldos contables post migración.
- Registro de logs y errores para trazabilidad y auditoría del proceso.

## Impacto logrado
Mi participación permitió una migración fluida y automatizada de grandes volúmenes de datos entre plataformas ERP, reduciendo tiempos operativos y asegurando la continuidad del 
negocio con cero pérdidas de información crítica.
